Features

Everything you need to automate your workflow

A powerful yet simple engine that runs entirely on your machine

Open Applications

Launch any application — browsers, editors, tools — as part of your automated workflow with a single step.

Run Commands

Execute shell commands, scripts, and system operations. Chain them together for complex automation pipelines.

Scheduled Triggers

Set up cron schedules, intervals, or login triggers. Your workflows run automatically — even when you forget.

100% Local & Private

No cloud, no telemetry, no accounts. Everything runs and persists on your device. Your data never leaves your machine.

Web Dashboard

A beautiful dark-themed UI to build, manage, and monitor workflows. Drag-and-drop steps, real-time logs, and keyboard shortcuts.

Open URLs

Open websites, documentation, or web apps automatically. Perfect for morning routines, standups, or project setups.

How It Works

Up and running in 60 seconds

Three simple steps to automate your Linux workflows

1

Install

Run a single command in your terminal. AutoFlow installs locally with zero external dependencies.

2

Build Workflows

Use the web dashboard or YAML files to create multi-step workflows with triggers and conditions.

3

Automate

Set schedules, run on login, or trigger manually. AutoFlow handles execution and logs everything.

CLI-First

Built for the terminal

Power users get a full CLI experience alongside the web dashboard

Terminal — autoflow
$ autoflow start
Starting AutoFlow on http://localhost:8000
 
$ autoflow list
Workflows dir: ~/.config/autoflow/workflows
 
[on] Morning Routine [cron] — Open browser and editor
morning.yaml | 3 steps
[on] Deploy Script [manual] — Deploy to production
deploy.yaml | 5 steps
 
$ autoflow run morning
